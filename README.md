# Vega Reservation System

A professional, high-end web application for managing hotel and property reservations. Built with PHP, PDO, and modern UI/UX principles.

## 🚀 Features

- **Premium Dashboard**: Real-time overview of reservation statistics.
- **Client Management**: Full list of guests with stay periods and contact details.
- **Dynamic Booking**: Intuitive form for quick guest registration.
- **World-Class UI**: Modern Dark Mode with glassmorphism, mesh backgrounds, and micro-animations.
- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile.
- **Secure Backend**: PHP PDO integration for robust database management.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Custom Variables, Grid, Flexbox), JavaScript.
- **Backend**: PHP 8.x.
- **Database**: MySQL (PDO).
- **Design Concepts**: Glassmorphism, Neon Accents, Mesh Gradients.

## 📦 Installation

1. Clone the repository to your local server (XAMPP/WAMP).
2. Import the database schema from the provided `.sql` file (or use `db.php` configuration).
3. Update `db.php` with your local credentials.
4. Access via `http://localhost/projet_final/`.
